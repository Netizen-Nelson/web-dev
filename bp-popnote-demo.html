② Panel 模式 — 觸發器綁定指定 <div>

每個觸發器加上 data-popover-panel="#panel-a"， 點擊後內容顯示在右側欄位而非彈窗。再次點擊同一觸發器會收起內容。

③ Global Panel 模式 — 全域設定一個共用欄位

透過 PopoverConfig.set({ panelTarget: '#global-panel' })， 頁面上所有觸發器都會顯示在同一處，無須逐一加上屬性。 使用 data-popover-panel="false" 可讓特定觸發器退回彈窗模式。

下方的觸發器未設置 data-popover-panel，但透過全域設定，仍使用面板模式。 「彈窗（退出全域）」那個例外地使用了 data-popover-panel="false"。

⑤ 屬性與 API 參考

觸發器屬性

屬性說明
data-popover-content行內 HTML 內容
data-popover-title標題（可同時出現）
data-popover-target內容來源的 CSS 選擇器（支援 <template>）
data-popover-panelPanel 目標 div 的 CSS 選擇器；設為 "false" 強制彈窗
data-popover-labelPanel 頂部來源標籤（省略則取觸發器文字）
data-popover-theme主題名稱（見下方色票）
data-popover-placementtop / bottom / left / right
data-popover-arrow設為 "false" 隱藏箭頭
data-popover-fontsize字體大小（如 0.9rem）
data-popover-maxwidth彈窗最大寬度（如 400px）
data-popover-offset觸發器與彈窗的間距（px）
data-popover-interval輪播間隔（ms），0 = 手動
data-popover-carousel-animslide / crossfade
data-popover-hint設為 "false" 隱藏底線提示

全域 API

// 全域設定
PopoverConfig.set({
  theme:       'dark',          // 預設主題
  panelTarget: '#my-notes',     // 全域 panel 目標 div
  maxWidth:    '560px',
  offset:      8,
  fontSize:    '1rem',
  placement:   'top',
  carousel: {
    animation: 'slide',         // 'slide' | 'crossfade'
    interval:  3000             // ms，0 = 手動
  }
});

// 自訂主題
PopoverConfig.addTheme('brand', {
  bg:          '#0d0d18',
  titleColor:  '#a0c4ff',
  borderColor: '#a0c4ff'
});
