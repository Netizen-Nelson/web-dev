屬性 API

外觀 / 資料

屬性預設值說明
color-labelshell軸標籤文字顏色（調色盤名稱或 hex）
color-webstone格線與軸線顏色
color-fillsky資料多邊形填色與頂點顏色
min-scale0最小刻度值
max-scale10最大刻度值
levels5格線層數（2–10）
show-valuestrue是否在標籤旁顯示數值
stroke-width2資料多邊形線寬（px）
web-width0.75格線與軸線粗細（px），數值越大網格越明顯
animatetrue是否啟用淡入動畫
title（無）圖表標題文字

字體大小

屬性預設值說明
root-font 14 基礎字體大小（px）。所有未單獨設定的文字皆依固定比例縮放：
標題 ×1.036  ·  標籤 ×0.964  ·  數值 ×0.786  ·  刻度 ×0.679。
接受純數字或含 px 單位（如 "18" 或 "18px"）。
font-title auto 標題字體大小（px）。設定後不受 root-font 影響。
font-label auto 軸標籤字體大小（px）。設定後不受 root-font 影響。
font-value auto 數值字體大小（px）。設定後不受 root-font 影響。
font-tick auto 刻度標籤字體大小（px）。設定後不受 root-font 影響。 
/* ① root-font 一次縮放全部文字 */
<bp-radar root-font="18" color-fill="sky">...</bp-radar>

/* ② 個別覆蓋，其餘仍依 root-font 比例 */
<bp-radar root-font="16" font-title="20" font-tick="8">...</bp-radar>

/* ③ 全域預設（優先序：元件屬性 > BpRadarConfig > 內建）*/
window.BpRadarConfig = {
  'root-font':  '18',   // 所有元件預設放大
  'font-value': '12',   // 數值欄固定 12px
  'color-fill': 'lavender',
};
