flow-card

圓形圖示卡片 Web Component  ·  無 Shadow DOM  ·  最多五欄

3 欄 · 主題色個別指定
Process is not only a single task; it is a high-level view of interrelated tasks done to produce a product or a service.

What To Do A particular set of work instructions or business rules that align with the agency's objectives, goals, and management philosophies.

Why Do It On the flip side of policy, procedures refer to detailed instructions on how to do tasks or perform a sub-process action.

How To Do It
4 欄 · connector-icon · root-font 16
不設限地蒐集各種可能性，鼓勵天馬行空，先求量再求質。 將概念轉換為具體的藍圖，定義使用者流程與視覺結構。 依據規格實作功能，保持敏捷節奏，快速迭代修正。 部署至正式環境，監測數據，持續優化使用者體驗。
5 欄 · 各欄獨立 connector 文字 · icon-size 64
需求分析與訪談，釐清核心問題與目標用戶。 架構規劃與技術選型，制定里程碑與時程表。 撰寫程式碼，單元測試，持續整合自動化流程。 QA 全面測試，效能調校，安全性稽核。 正式上線，文件交付，SLA 維運支援。

屬性 API

<flow-card> 容器屬性

屬性預設值說明
themesky全域預設圖示圓圈顏色，可被個別 flow-content 的 icon-color 覆蓋
connectorVS連結徽章的文字，可被個別 flow-content 的 connector 屬性覆蓋
connector-icon（無）連結徽章改用 Bootstrap icon，例如 bi-arrow-right-fill，優先於 connector 文字
connector-colorsalmon連結徽章背景色
connector-size36連結徽章直徑（px）
connector-overlap0.1徽章壓入兩側卡片的比例（相對於 connector-size）。0.1 = 各壓入 10%；0 = 不重疊；0.5 = 各壓入半徑
icon-size80頂部圖示圓圈直徑（px）
icon-font32圓圈內 Bootstrap icon 的字體大小（px）
icon-shadowtrue圖示圓圈是否有陰影
card-bgregion卡片背景色（調色盤名稱或 hex）
card-radius12卡片圓角（px）
card-padding20卡片內距（px），上方會自動留出圖示圓圈的重疊空間
text-aligncenter卡片文字對齊方向：center / left / right
root-font14基礎字體（px），未個別設定時標題 ×1.14、內文 ×0.93 自動推算
font-titleauto標題字體大小覆蓋值（px）
font-bodyauto內文字體大小覆蓋值（px）
font-connectorauto連結徽章文字大小覆蓋值（px）
animatetrue卡片進場動畫（依序淡入）

<flow-content> 子元素屬性

屬性說明
iconBootstrap icon class，例如 bi-diagram-3-fill
icon-color此卡片圖示圓圈顏色，覆蓋父元素 theme
title卡片標題，顏色自動跟隨圖示圓圈顏色
connector此卡片左側連結徽章文字（覆蓋父元素 connector），設為空字串可隱藏文字
connector-icon此卡片左側連結徽章 Bootstrap icon（覆蓋父元素 connector-icon）
（內容）卡片主體內容，支援完整 HTML（按鈕、連結、圖片等）
/* 全域預設值 */
window.FlowCardConfig = {
  'theme':          'lavender',
  'connector':      '→',
  'root-font':      '16',
  'card-bg':        '#1e201e',
};

/* 基本 3 欄 */
<flow-card theme="sky" connector="VS" connector-color="salmon">
  <flow-content icon="bi-diagram-3-fill" title="Process">
    說明文字，<b>支援 HTML</b>，亦可放按鈕。
  </flow-content>
  <flow-content icon="bi-book-fill" icon-color="warning" title="Policy">
    ...
  </flow-content>
</flow-card>

/* 各欄獨立 connector 文字 */
<flow-content connector="01" connector-icon="bi-arrow-right-fill">...</flow-content>
