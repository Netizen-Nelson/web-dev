list-card.js

使用說明 · v1.0 · 無 Shadow DOM · 單一 JS 檔案

01 · 載入

在 </body> 前引入一個 JS 檔，無任何外部相依。

<script src="list-card.js"></script>
02 · 基本結構

初始只顯示第一張卡片，點擊虛線觸發器展開下一張，最後一張沒有虛線。 

<list-card>
  <card-item>第一張</card-item>
  <card-item>第二張</card-item>
  <card-item>第三張（最後，無虛線）</card-item>
</list-card>

第二張

展開後顯示。虛線仍可繼續點擊。

第三張

最後一張，沒有虛線。

04 · <card-item> 屬性
屬性說明預設值
data-width 固定寬度（flex: 0 0 <width>），不參與彈性分配 無
flex 彈性佔比；data-width 存在時忽略 1
padding 卡片內距，接受 CSS padding 值 10px 14px
bg 個別覆寫背景色 依 theme
align 內容對齊：start / center / end start
data-text-source 指定同頁 div 的 id，匯入其 innerHTML 作為內容 無

  <card-item flex="2">主要內容（佔 2 份）</card-item>
  <card-item>次要內容（佔 1 份）</card-item>
</list-card>

06 · data-text-source

指定同頁任意 div 的 id，元件會讀取其 innerHTML 作為卡片內容。 來源 div 不會被移動或刪除，可自行設為 display:none。 若 id 找不到，退回使用標籤間的 innerHTML。 

<list-card theme="orange">
  <card-item>行內撰寫的內容</card-item>
  <card-item data-text-source="my-div"></card-item>  <!-- 從 #my-div 匯入 -->
  <card-item>第三張</card-item>
</list-card>

<!-- 來源 div，可放頁面任意位置 -->
<div id="my-div" style="display:none">
  <p>這裡支援完整 HTML</p>
</div>
實際效果

行內內容

直接寫在 <card-item> 標籤之間。

從 #src-profile 匯入

此內容定義在頁面其他位置的 div，元件只讀取 innerHTML，不影響原始 div。

從 #src-skills 匯入

  • 來源 div 可設 display:none
  • 支援完整 HTML 標籤
  • 可用 .refreshSource() 重新同步
07 · JavaScript API

透過元素實例呼叫，需在元件載入後執行。

方法說明
.expandTo(n)展開至第 n 張（1-based）
.expandAll()展開全部
.reset()收回至只顯示第一張
.setTheme(name)動態切換主題
.getItems()回傳所有 card-item 節點陣列
.refreshSource(index?)重新同步 data-text-source；省略 index 則全部同步
.setConfig(obj)動態更新全域設定
// 取得元素
const card = document.getElementById('my-card')

card.expandTo(2)           // 展開到第二張
card.expandAll()            // 全部展開
card.reset()                // 收回
card.setTheme('pink-fill') // 動態切換主題
card.refreshSource(1)      // 重新同步第二張（index 0-based）

08 · DOM 事件
事件觸發時機event.detail
card-ready 元件初始化完成 { count, theme }
card-expand 點擊虛線展開新卡片 { index, total } 
document.getElementById('my-card')
  .addEventListener('card-expand', e => {
    console.log(`展開第 ${e.detail.index + 1} 張，共 ${e.detail.total} 張`)
  })
09 · 全域設定（window.ListCardConfig）

在載入 list-card.js 之前設定，作為所有元件的預設值。 優先順序：行內屬性 > ListCardConfig > 元件預設值。 

window.ListCardConfig = {
  theme        : 'safe',      // 預設主題
  radius       : '6px',
  itemPadding  : '10px 14px',
  dividerWidth : '16px',
  animDuration : '280ms',
  animEasing   : 'ease-out',
}

<script src="list-card.js"></script>
