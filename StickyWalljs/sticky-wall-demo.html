📌 StickyWall 便利貼元件示範

JS Web Component · No Shadow DOM · Zero Dependencies

在任何網頁上
雙擊建立便利貼

只要一個 <script> 標籤，雙擊頁面空白處即可建立可拖曳、可調色、自動儲存的便利貼。

dblclick 任意空白處 → 建立便利貼

掛載與設定方式

<!-- 1. 可選：載入前定義全域設定 -->
<script>
  window.StickyNotesConfig = {
    defaultColor : '#81E6D9',   // 初始顏色
    defaultWidth : 300,          // 初始寬度 px
    position     : 'fixed',       // 'fixed' | 'absolute'
    storageKey   : 'my_notes',    // localStorage 鍵名
    excludeClass : 'no-sticky',  // 此 class 的元素不觸發
    colors: [                      // 完全自訂調色盤（選填）
      { name:'自訂紫', val:'#9B59B6' },
    ],
  };
</script>

<!-- 2. 載入元件（放 </body> 前） -->
<script src="sticky-wall.js"></script>

<!-- 3. 公開 API -->
<script>
  StickyWall.clearAll();   // 清除全部
  StickyWall.export();     // 匯出資料陣列
  StickyWall.import(arr); // 匯入資料陣列（疊加）
  StickyWall.save();       // 手動觸發儲存
</script>

元件功能一覽

  • 🖱️
    雙擊建立 頁面任意空白處 dblclick，便利貼出現在游標位置。
  • 🎨
    11 種品牌色 標題列色點即時切換，顏色對應完整品牌調色盤。
  • ✋
    拖曳移動 拖曳標題列即可任意定位，點擊自動置頂。
  • ↔️
    右下角縮放 CSS resize:both，拖曳右下角自由調整大小。
  • ─
    收合 / 展開 點「─」只保留標題列，再點展開全文。
  • 💾
    自動持久化 所有操作即時寫入 localStorage，重開頁面自動還原。
  • 🌐
    全域設定系統 StickyNotesConfig 可在載入前覆蓋任意預設值。
  • ♿
    語義化 HTML article / header / nav / section 語義標籤，支援鍵盤操作。
  • 📤
    匯出 / 匯入 StickyWall.export() 取得 JSON，import() 批次載入。
  • 🚫
    排除區域 excludeClass 指定不觸發的區塊，避免誤操作。
