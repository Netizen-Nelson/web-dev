MVC（Model-View-Controller）是一種將應用程式分為三個核心層的架構模式。

  • Model：資料與商業邏輯
  • View：使用者介面
  • Controller：協調 Model 與 View

SPA（Single Page Application）在初次載入後不再重新整理整個頁面，而是透過 JS 動態更新 DOM，帶來接近原生 App 的操作體驗。

SSR（Server-Side Rendering）在伺服器端將 HTML 完整渲染後再傳送給瀏覽器，有助於 SEO 與首屏載入速度。

相關：Next.js、Nuxt.js、SvelteKit 均支援 SSR。

CDN（Content Delivery Network）透過分散在各地的伺服器節點，讓使用者從距離最近的節點取得靜態資源，降低延遲。

WebAssembly 是一種低階二進位指令格式，可在瀏覽器中以接近原生速度執行 C/C++/Rust 等語言編譯的程式。

PWA（Progressive Web App）利用 Service Worker 與 Web App Manifest，讓網頁具備離線運作、推播通知、安裝到桌面等原生 App 特性。

將 bp-tools.js 推送到 GitHub，透過 jsDelivr CDN 在任何網頁中引用即可。支援版本號鎖定（@v1.0.0）。
