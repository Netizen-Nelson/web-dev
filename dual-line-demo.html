Web Component · dual-line.js v1.2.0

<dual-hr> ＋ <dual-vr>

雙色水平線與垂直線元件。支援 solid／dashed／dotted 三種樣式，全域配置系統，零依賴。

水平線 <dual-hr>

solid — yellow : sky = 3:7，1px（預設）
solid — lavender : safe = 1:1，3px
solid — warning : stone = 7:3，2px
solid — orange : sky = 2:8，4px
solid — 自訂 hex：#ff3e80 : #00e5b0 = 5:5，2px
dashed — yellow : sky = 3:7，1px，dash=8px gap=5px（預設）
dashed — lavender : orange = 1:1，3px，dash=10px gap=6px
dashed — warning : safe = 7:3，2px，細密 dash=4px gap=3px
dashed — pink : info = 4:6，4px，寬鬆 dash=16px gap=8px
dotted — yellow : sky = 5:5，4px（圓點建議 thickness ≥ dash）
dotted — salmon : lavender = 3:7，8px，大圓點
dotted — safe : orange = 6:4，6px
dotted — 細線 2px（退化為短方塊）— sky : pink = 1:1

全域配置 API

呼叫 DualLine.configureHR() 後所有未帶屬性的 <dual-hr> 立即更新 
// 設定 <dual-hr> 全域預設
DualLine.configureHR({
  colorLeft:  'salmon',
  colorRight: 'stone',
  ratio:      '6:4',
  thickness:  '2px',
  lineStyle:  'dashed',
  dash:       '8px',
  gap:        '5px',
});

// 設定 <dual-vr> 全域預設
DualLine.configureVR({
  colorTop:    'pink',
  colorBottom: 'info',
  thickness:   '2px',
  height:      '100px',
});

// 同時更新兩者（只更新共用屬性如 lineStyle）
DualLine.configure({ lineStyle: 'dotted', dash: '4px', gap: '4px' });

// 新增自訂色票
DualLine.addColor('brand', '#3d9970');
// 之後即可使用 color-left="brand"

屬性速查

<dual-hr> 水平線

屬性說明預設
color-left左段色彩（色票名稱或 CSS 色碼）yellow
color-right右段色彩sky
ratio左:右寬度比，如 3:75:5
thickness線條高度1px
margin上下外距8px 0
line-stylesolid ｜ dashed ｜ dottedsolid
dash虛線段長度8px
gap虛線段間距5px

<dual-vr> 垂直線

屬性說明預設
color-top上段色彩（可用 color-left 別名）yellow
color-bottom下段色彩（可用 color-right 別名）sky
ratio上:下高度比5:5
thickness線條寬度1px
height線條總高度80px
margin左右外距0 8px
line-stylesolid ｜ dashed ｜ dottedsolid
dash虛線段長度8px
gap虛線段間距5px
