Section 07
data-* 屬性完整參考
所有屬性均可在容器或透過 BPStepper.defaults 設定全域值。
data-* 屬性 CSS 變數 預設值 說明
data-theme—null主題名稱：lavender / sky / safe / special / warning / salmon / stone / pink / orange
data-connector—solid連接線樣式：solid / dashed / dotted
data-layout——版型：vertical（縱向）
data-auto-number—true自動注入步驟號徽章，設 false 關閉
data-color--bps-color#C3A5E5主強調色（邊框、連接線、徽章輪廓）
data-color-active--bps-color-active#C8DD5A進行中步驟色
data-color-done--bps-color-done#40c99a已完成步驟色
data-color-error--bps-color-error#F08080錯誤步驟色
data-text-color--bps-text#c6c7bd卡片說明文字顏色
data-title-color--bps-title#C3A5E5卡片標題顏色
data-badge-bg--bps-badge-bgrgba(...)步驟號徽章背景
data-card-bg--bps-card-bgrgba(...)卡片背景
data-card-bg-active--bps-card-bg-actrgba(...)進行中卡片背景
data-card-bg-done--bps-card-bg-donergba(...)完成卡片背景
data-card-bg-error--bps-card-bg-errrgba(...)錯誤卡片背景
data-stroke--bps-stroke2px邊框 / 連接線粗細
data-radius--bps-radius12px卡片圓角
data-card-width--bps-width220px卡片寬度
data-card-padding--bps-padding16px 18px卡片內距（支援 CSS 縮寫）
data-gap--bps-gap36px步驟間距（同時控制連接線長度）
data-badge-size--bps-badge-sz28px步驟號徽章大小
data-font-size--bps-fs0.9rem說明文字大小
data-title-size--bps-title-fs1rem標題文字大小
data-badge-font-size--bps-badge-fs0.82rem步驟號大小
data-arrow-size--bps-arrow6px箭頭尺寸
