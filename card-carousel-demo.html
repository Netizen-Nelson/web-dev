Section 02
Fill vs Invert — 相同主題，雙樣貌
invert：深底色 + 主題色邊框/文字。 fill：主題色底色 + sky (#08a9d1) 邊框/文字。

Section 03
單張覆寫主題 & 自訂圖示尺寸
每張 <carousel-card> 可各自設定 theme / variant， 覆蓋父層設定。右側展示大圖示版本（icon-size="3rem"）。

Section 04
動態新增 / 移除卡片
透過 JS 操作 DOM 即可新增或移除 <carousel-card>， MutationObserver 自動偵測並更新。
Section 05
使用方式
引入 Bootstrap Icons CSS 與 card-carousel.js 即可。
HTML — 基本使用
<!-- 1. 引入相依 --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css"> <script src="card-carousel.js"></script> <!-- 2. 宣告元件 --> <card-carousel theme="lavender" variant="invert" card-width="420px" icon-size="2.2rem" visible-back="2"> <carousel-card icon="bi-person-fill" name="Gillian Gardner" sub="New York City, NY"> </carousel-card> <!-- 單張覆寫主題 --> <carousel-card icon="bi-star-fill" name="Fabio Franklin" sub="Jacksonville, FL" theme="sky" variant="fill"> </carousel-card> </card-carousel>
JS — 全域設定 & 動態操作
// 覆寫全域預設（頁面載入時執行） CardCarousel.config({ iconSize: '2.6rem', iconPadding: '16px', iconRadius: '50%', // 圓形圖示 cardWidth: '480px', cardRadius: '20px', visibleBack: 3, transitionMs: 400, stackOffsetY: 30, stackScale: 0.06, stackBlur: 2, stackOpacity: 0.25, }); // 取得品牌色表 console.log(CardCarousel.tokens.lavender); // #C3A5E5 // 動態新增卡片 const cc = document.querySelector('card-carousel'); const newCard = document.createElement('carousel-card'); newCard.setAttribute('icon', 'bi-gem'); newCard.setAttribute('name', 'New Person'); newCard.setAttribute('sub', 'Chicago, IL'); cc.appendChild(newCard); // 自動偵測並更新 // 即時切換主題 cc.setAttribute('theme', 'sky'); cc.setAttribute('variant', 'fill');
Section 06
屬性一覽
HTML 屬性可覆寫全域預設；CardCarousel.config() 設定對所有元件生效。
屬性 (HTML attr) 預設值 說明
themelavender主題色名稱，對應品牌色 Token
variantinvertinvert（深底）或 fill（色底）
card-width420px元件整體寬度
icon-size2.2remBootstrap Icon 字型大小
icon-radius12pxIcon 槽圓角（設 50% 可做圓形）
visible-back2後方顯示幾張堆疊卡片
CardCarousel.config() 鍵 預設值 說明
transitionMs370動畫時長（毫秒）
stackOffsetY27每層向上位移（px）
stackScale0.07每層縮放遞減量
stackBlur1.8每層模糊遞增量（px）
stackOpacity0.22每層透明度遞減量
iconPadding13pxIcon 槽內距（影響槽大小）
cardRadius18px卡片圓角
