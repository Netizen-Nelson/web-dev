⟳ rolling-number.js

無 Shadow DOM 的跳動數字 Web Component · v1.1

最簡用法

只需 value 屬性
<rolling-number value="9527"></rolling-number>

字型控制

font-family + font-size + font-weight
<rolling-number value="3721" font-family="'Georgia', serif" font-size="2.5rem" color="#C3A5E5"> </rolling-number>

金融數字（前後綴 + 小數）

prefix / suffix / separator / decimals
<rolling-number value="1234567.89" prefix="NT$" suffix=" 元" decimals="2" font-size="2.2rem" easing="expo"> </rolling-number>

緩動預設值比較（easing）

6 種內建緩動
smooth（預設）
expo
decel
bounce
elastic
linear
<rolling-number value="8899" easing="bounce"></rolling-number> <!-- smooth | expo | decel | bounce | elastic | linear --> <!-- 或自訂：easing="cubic-bezier(0.34,1.56,0.64,1)" -->

色彩模式（color-scheme）

rainbow（預設）—— 每位數字不同色
mono —— 全部同色（mono-color 指定色票）
color —— 直接指定單一顏色（覆蓋 color-scheme）

全域預設覆蓋（RollingNumber.defaults）

統一設定所有元件的預設值，不必每個都寫屬性
// 在載入 rolling-number.js 後執行 RollingNumber.defaults.fontSize = '2rem'; RollingNumber.defaults.easing = 'expo'; RollingNumber.defaults.colorScheme = 'mono'; RollingNumber.defaults.monoColor = '#40c99a'; RollingNumber.defaults.duration = 2.0; // 之後建立的元件都會套用上面的預設 const el = document.createElement('rolling-number'); el.setAttribute('value', '12345'); container.appendChild(el);

互動：動態更新

修改數值後點「播放」，元件重新執行動畫
// JS 方式更新 const el = document.getElementById('live'); el.setAttribute('value', '12345'); // → 自動重新播放 el.setAttribute('easing', 'bounce'); // 或使用封裝的快捷方法 el.countTo(99999);

滾動觸發（autoplay / threshold）

autoplay="true"（預設）：進入視窗才播放；autoplay="false"：立即播放
autoplay（進視窗觸發）
autoplay="false"（立即播）
<!-- 進入視窗才播，20% 可見面積觸發 --> <rolling-number value="777777" autoplay="true" threshold="0.2"></rolling-number> <!-- 頁面載入立即播 --> <rolling-number value="123456" autoplay="false"></rolling-number>

屬性速查表

屬性 型別 預設 說明
value number 0 目標數值（必填，支援小數）
font-family string inherit 字型名稱
font-size string 3rem 字型大小（rem / px）
font-weight string 800 字重
color string '' 指定單色，覆蓋 color-scheme
color-scheme string rainbow rainbowmono
mono-color string #C8DD5A color-scheme=mono 時的顏色
separator-colorstring #c6c7bd 逗號、小數點、前後綴的顏色
duration number 1.6 動畫時長（秒）
delay-step number 0.08 每欄錯開延遲（秒）
easing string smooth smoothexpodecelbounceelasticlinear 或 cubic-bezier(…)
separator string , 千位分隔符，空字串=不顯示
decimals number 0 顯示小數位數
prefix string '' 前綴文字，如 NT$
suffix string '' 後綴文字，如 元、%
autoplay boolean true 進入視窗才播（IntersectionObserver）
threshold number 0.2 觸發閾值（0–1），越高越晚觸發
line-height number 1.35 每格高度係數（相對於 font-size）

JavaScript API

const el = document.querySelector('rolling-number'); // 播放動畫 el.play(); // 更新數值並重播 el.countTo(99999); el.setAttribute('value', 99999); // 同效果 // 全域預設（影響之後建立的所有元件） RollingNumber.defaults.easing = 'expo'; RollingNumber.defaults.fontSize = '2.5rem'; RollingNumber.defaults.duration = 2.0; // 查看內建緩動與色彩 console.log(RollingNumber.EASINGS); console.log(RollingNumber.RAINBOW);
